# RAG Simulation for LLM Context

This example demonstrates the core concept of Retrieval-Augmented Generation (RAG) for Large Language Models (LLMs). It simulates how an LLM can use external, domain-specific information to answer queries accurately and reduce the risk of generating incorrect or 'hallucinated' responses. The script showcases a simple retrieval mechanism and an LLM simulation that incorporates the retrieved context into its output.

## Language

`python`

## How to Run

1. Ensure you have Python 3 installed.
2. Save the code as `main.py`.
3. Run from your terminal: `python main.py`
